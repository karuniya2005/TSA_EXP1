# PROGRAM:
```
import pandas as pd
import matplotlib.pyplot as plt
df = pd.read_csv("/content/train.csv")
df = df.sort_values(by='ram')
plt.plot(df['ram'], df['price_range'], marker='o')
plt.title("Mobile Price Range vs RAM")
plt.xlabel("RAM (MB)")
plt.ylabel("Price Range")
plt.show()
```
